Autoryzacja GitHub

Wklej Fine-grained Personal Access Token z github.com/settings/tokens.
Wymagane uprawnienia (Account permissions): Copilot requests — write, Models — read, Plan — read.

Dostępne modele

Modele z Copilot SDK i GitHub Models API. Plan Copilot wpływa na limity.

Zapisz token w sekcji Authorization, aby załadować modele.

Oczekujące zatwierdzenia

Gdy agent chce wywołać usługę HA, uruchomić komendę shell lub zmodyfikować plik, akcja trafia tutaj do zatwierdzenia.
