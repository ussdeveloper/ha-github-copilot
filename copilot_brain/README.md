# Copilot Brain Add-on

Experimental Home Assistant add-on for a GitHub-powered assistant with:

- Ingress UI
- split chat + HA terminal workspace
- GitHub Models integration
- GitHub App authentication
- guarded Home Assistant tool execution
- MCP support for GitHub Copilot

## Current state

This add-on is **in progress** and currently published as an **experimental** build path from repository source.

## Important

- Do not commit real secrets.
- Do not commit `.env`, PEM keys, tokens, `/data`, or `.data` contents.
- Configure credentials from the built-in Web UI, or use local untracked files only for local development.

See `DOCS.md` for setup and usage details.
